# Data Structures in C

This repository contains implementations of basic data structures and algorithms using C.

## Topics Covered
- Arrays
- Stack, Queue, Deque
- Linked Lists
- Trees & Heaps
- Graphs (BFS, BFT, DFS, DFT)
- Sorting Algorithms
- Recursion
- Union-Find

## Language Used
- C Programming Language

## Purpose
This repository is created for practice, understanding core concepts,
and academic preparation.
